# Riwas Thapa — AI Developer Portfolio

A modern, clean portfolio website showcasing AI/ML projects, skills, and experience. Built with vanilla HTML, CSS, and JavaScript.

## 🎯 About

AI Developer focused on LLM-powered applications, RAG systems, and production ML. I turn complex AI research into tools that work reliably in the real world.

- **Location**: Bangalore
- **Focus**: LLM applications, RAG pipelines, Computer Vision
- **Status**: Open to AI/ML roles

## 🚀 Features

- **Responsive Design**: Mobile-first approach with smooth responsive breakpoints
- **Modern Aesthetics**: Clean, minimalist design with smooth animations
- **Scroll Animations**: Fade-in effects as you scroll through sections
- **Dark Mode Ready**: CSS custom properties for easy theming
- **Performance Optimized**: Vanilla JS, no heavy frameworks
- **Accessibility**: Semantic HTML, proper contrast ratios

## 📂 Sections

1. **Hero** - Introduction with availability status and key stats
2. **Projects** - Detailed project cards with metrics and tech stack
3. **Skills** - Categorized skill groups (ML, Backend, Data)
4. **About** - Personal background and highlights
5. **Experience** - Timeline of professional roles
6. **Contact** - Direct email link and social connections

## 💻 Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Fonts**: Inter (Google Fonts)
- **Icons**: Unicode/Emoji
- **Performance**: No external dependencies
